# mini_drone_brain-

A beginner-level autonomous drone simulation developed with Python and Pygame.

## Project Overview

This project simulates a simple autonomous drone operating in a grid-based environment. The drone starts from a predefined base location and can navigate through the map while avoiding obstacles.

The purpose of this project is to build a strong foundation in:

- Autonomous system logic
- Path planning concepts
- Grid-based navigation
- Object-Oriented Programming (OOP)
- Simulation development with Pygame

This project is being developed step by step and will continue to evolve with additional autonomous capabilities.

## Current Features

- Grid-based environment
- Base and mission locations
- Obstacle placement
- Drone movement using keyboard controls
- Collision prevention with obstacles
- Boundary checking
- Object-oriented project structure
